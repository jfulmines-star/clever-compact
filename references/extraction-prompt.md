# Clever Compact — Extraction Prompt

Run this against your recent session history to extract what matters before a compact.

---

Review the session history above and extract the following. Be concise — this is a navigation aid, not a transcript.

## Extract:

**1. Active Workstreams**
For each thing currently in progress or blocked:
- Name of the workstream
- Status: done / in-progress / blocked / waiting
- If blocked: what is the blocker and who resolves it
- If done: URL or file path of the artifact
- If waiting: waiting on what / whom

**2. Open Tasks**
List every explicit action item that has NOT been completed. Include owner if known.

**3. Key Decisions**
Decisions made this session that future sessions need to honor. Include brief rationale if it's not obvious.

**4. Credentials & IDs Captured**
Any API keys, project IDs, account IDs, tokens, or config values mentioned this session.
If already saved to TOOLS.md, write "see TOOLS.md" instead of repeating them.

**5. Relationship Context**
Anything said about people (clients, partners, team members) that would affect how to interact with them next session.

**6. Remember Flags**
Anything the user explicitly said to remember, flag, or not forget.

---

Format the output using the state-template.md structure.
